    使用

    • 软件用于收藏用户自己的评论, 用户有多个账号也可以使用

    功能

    • 仅获取文字
    • 获取后, 追加内容不会获取

    细节

    • 不要暴力测试 / 会被三方封 IP
    • 可以本地搜索到最新的和最老的评论, 来检测是否完整
    • 删除用户数据时注意 (清除数据后无法恢复, 有些收藏会永久消失)

    私密或安全

    • 涉及到数据保密请自己负责
    • 软件不会上传用户数据, 所有数据保存在本地
    • 升级请备份数据, 直接打包 (rar, zip) 安装目录即备份
      %LOCALAPPDATA%\Fisoft\CommentDroplet
